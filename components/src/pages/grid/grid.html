Grid

Ionic’s grid system is based on flexbox, a CSS feature supported by all devices that Ionic supports. The grid is composed of three units — grid, rows and columns. Columns will expand to fill their row, and will resize to fit additional columns. Rows have no padding or margin applied to them. If that is needed, ion-grid can be added to provide additional padding. In order to set width use width attribute on ion-col tag.
