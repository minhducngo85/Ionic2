Install
npm config set proxy http://username:password@proxy:port
npm install -g ionic cordova
Code example:
set PROXY=http://username:password@proxy:port
ionic start cutePuppyPics --v2
cd cutePuppyPics
ionic serve or ionic lab
Generate page:
Ionic g page detail
 
 
Clone from Git
Clone this repository: git clone https://github.com/driftyco/ionic-conference-app.git.
Want to use TypeScript? Both the master branch and the typescript branch now use TypeScript.
Run npm install from the project root.
Install the ionic CLI (npm install -g ionic)
Run ionic serve in a terminal from the project root.
